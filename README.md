# InternshipX

Launch Your Career

InternshipX is a full-stack internship management platform built with Node.js, Express.js, MongoDB Atlas, HTML, CSS, and JavaScript.

## Features

- Student & Company Authentication
- Gmail OTP Verification
- JWT Authentication
- Internship Management
- Application Tracking
- Role-Based Access Control

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- HTML
- CSS
- JavaScript
